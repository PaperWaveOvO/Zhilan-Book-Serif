# 纸澜书宋

一款衍生于霞鹜新致宋的中文宋体字体。

尝试在霞鹜新致宋的基础上做一些微调，希望能更适合书籍排版和试卷印刷。目前还在起步阶段，由两个字体爱好者共同打磨。

# Zhilan Book Serif

A Chinese Serif font derived from LXGW Neo ZhiSong.

A humble attempt to refine LXGW Neo ZhiSong for better book and exam paper layout. Still a work in progress by two font enthusiasts.
